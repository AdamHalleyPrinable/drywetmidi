    Class UnknownChunkException

    The exception that is thrown when the reading engine encountered a chunk with unknown ID.
    Inheritance
    Object
    MidiException
    UnknownChunkException
    Namespace: Melanchall.DryWetMidi.Core
    Assembly: Melanchall.DryWetMidi.dll
    Syntax
    public sealed class UnknownChunkException : MidiException
    Remarks
    Note that this exception will be thrown only if UnknownChunkIdPolicy is set to Abort for the ReadingSettings used for reading a MIDI file.
