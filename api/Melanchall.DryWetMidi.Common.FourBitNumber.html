    Type that is used to represent a four-bit number (0-15).
    Implements
    IComparable<FourBitNumber>
    IConvertible
    Namespace: Melanchall.DryWetMidi.Common

    Syntax
    public struct FourBitNumber : IComparable<FourBitNumber>, IConvertible
    Remarks
    Four-bit numbers widely used by MIDI protocol as parameters of MIDI events. So instead of manipulating built-in C# numeric types (like byte or int) and checking for out-of-range errors all validation of numbers in the [0; 15] range happens on data type level via casting C# integer values to the FourBitNumber.
