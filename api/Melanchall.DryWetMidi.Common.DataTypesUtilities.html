    GetVlqBytes(Int32)

    Gets bytes of a number in variable-length quantity (VLQ) format.
    Declaration
    public static byte[] GetVlqBytes(this int number)
    Parameters
    Type Name Description
    Int32 number Number to get VLQ bytes for.
    Returns
    Type Description
    Byte[] Bytes representing the number coded in VLQ.
    Remarks
    Numbers in VLQ format are represented 7 bits per byte, most significant bits first. All bytes except the last have bit 7 set, and the last byte has bit 7 clear. If the number is between 0 and 127, it is thus represented exactly as one byte.
