    ReadingHandler.TargetScope scope Scope of MIDI data to handle.
    Remarks
    It's important to set desired scope to avoid performance degradation. For example, if you want to handle start of file reading and start of track chunk reading, specify scope to TargetScope.File | TargetScope.TrackChunk.
