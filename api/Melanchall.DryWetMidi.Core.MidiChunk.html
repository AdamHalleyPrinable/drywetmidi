    Remarks
    MIDI Files are made up of chunks. Each chunk has a 4-character ASCII string ID and a 32-bit length, which is the number of bytes in the chunk. This structure allows future chunk types to be designed which may be easily be ignored if encountered by a program written before the chunk type is introduced. The length of the chunk refers to the number of bytes of data which follow (the eight bytes of ID and length are not included).Therefore, a chunk with a length of 6 would actually occupy 14 bytes in the file.
